# Rental Car

## Short Description

The **Rental Car** project is a web application for searching and renting cars. Users can browse the car catalog, filter by brand, price, mileage, and add cars to their favorites.

## Key Features

- Browse a catalog of cars with information about year, brand, model, mileage, and price.
- Filter cars by brand, price, mileage, and type.
- Add or remove cars from favorites.
- For desktop devices.
- Detailed page for each car with full information.

## Technical Details

- Next.js (App Router)
- TypeScript
- Zustand for managing favorites state
- Next/Image for image optimization
- CSS modules for component styling

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/OleksandrRiabokon91/rental-car
   ```

2. Navigate to the project folder:

   ```bash
   cd rental-car
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm run dev
   ```

5. Open in your browser:

   ```
   http://localhost:3000
   ```

## Usage

- Go to the homepage to see the hero section and recommended cars.
- Use filters to search for cars by specific criteria.
- Click the heart icon to add a car to your favorites.
- Visit the car detail page for full information about each car.
